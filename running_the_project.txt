Informações úteis.

Navegue até o ...\eng-software\diabetter\backend
Nesse diretório, executar o comando gradle apropriado para o seu SO para buildar o projeto.
Uma vez que o projeto esteja buildado, para rodar o BD H2, siga estes passos:

1- no terminal, digite: gradlew bootRun  (roda o servidor Spring Boot)
2- Acessar a seguinte URL: http://localhost:8080/h2-console
3- usar estas credenciais:
    JDBC URL: jdbc:h2:mem:diabetter
    User: sa
    Password: password
